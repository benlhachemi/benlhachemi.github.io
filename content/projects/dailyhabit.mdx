### **Project Summary**

- **Date:** May, 2026
- **Description:** DailyHabit is a beautiful, feature-rich habit tracking application designed to help users build and maintain daily habits through an intuitive interface and powerful analytics.

### Motivation

This is a project to practice building apps with AI Coding Agents while also promoting Root UI.

### Links

Live demo: https://daily-habit-umber.vercel.app/

GitHub repo: https://github.com/benlhachemi/daily-habit

YouTube tutorial: https://youtu.be/U6cd310jHC8

### **Tech Stack**

- **Framework:** Next.js 16.2.4 (App Router) + React 19.2.4 + TypeScript 5
- **Styling:** Tailwind CSS 4 (OKLCH), shadcn 4.5.0, **`@base-ui/react`** primitives
- **Database:** RxDB (reactive local database, offline-first)
- **Animations:** Motion, Lottie React, canvas-confetti
- **Charts:** Recharts
- **Tooling:** Bun, ESLint, PostCSS

### **Key Features**

- **Habit Management:** Create custom habits with goals, units, emojis, and 11 color options; daily tracking with satisfying interactions; archive system
- **Analytics Dashboard:** Streak tracking, completion rates, progress charts (Recharts), historical trends
- **UX:** Dark/light theme (next-themes), responsive design, keyboard navigation, celebration animations on completion
- **Data Privacy:** 100% local storage via RxDB — no accounts, no cloud, fully offline

### **What I Learned**

- How to build apps using AI Coding Agents
- Reactive database design with RxDB (real-time data streams, local-first architecture)
- Next.js 16 App Router patterns (layout system, route groups)
- CSS-first Tailwind 4 configuration with OKLCH color space
- shadcn 4 with **`@base-ui/react`** primitives (not Radix)
- Advanced animation orchestration (Motion stagger, confetti celebrations)
- Building accessible, keyboard-navigable UIs with hotkeys
- Mobile-first responsive design with a component-driven architecture
