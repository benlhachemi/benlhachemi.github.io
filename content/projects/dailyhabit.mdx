### **Project Summary**

- **Date:** March, 2026
- **Description:** DailyHabit is a comprehensive platform built to streamline and separate concerns in modern web applications. It demonstrates clean architecture patterns and robust backend design.

### Motivation

This project was built to explore domain-driven design patterns and practice building scalable applications with NestJS.

### Links

GitHub repo: https://github.com/benlhachemi/comsep

### **Tech Stack**

- **Framework:** NestJS + Next.js + TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Database:** PostgreSQL with Prisma ORM
- **Authentication:** JWT-based auth
- **Tooling:** ESLint, Prettier

### **Key Features**

- **Separation of Concerns:** Clean modular architecture with distinct service layers
- **RESTful API:** Well-designed endpoints with proper error handling
- **Responsive UI:** Mobile-first design system
- **Authentication:** Secure user authentication and authorization

### **What I Learned**

- Domain-driven design in practice
- NestJS module architecture and dependency injection
- Building maintainable API layers
- Prisma ORM patterns and database design
