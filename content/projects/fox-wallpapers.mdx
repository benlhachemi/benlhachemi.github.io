### **Project Summary**

- **Date:** Mar, 2025
- **Description:** Windows Live Wallpaper Engine built using TauriJs (React + Rust).

### Motivation

The goal was to learn how to build windows apps and publish to the microsoft app store.

### Links

Microsoft App Store page: https://apps.microsoft.com/detail/9n7r9zf4vsx7

### **Tech Stack**

- TauriJs
- React
- Rust

### **Key Features**

- Diverse Collection - Explore a variety of high-quality animated wallpapers.
- Instant Apply - One-click setup for hassle-free customization.
- Smooth Performance - Optimized for minimal system impact.

### **What I Learned**

- Rust basics
- How to package a bundled Tauri app (.exe) to MSIX (using MSIX Packaging Tool) and publish it to the Microsoft App Store.
- How to create a Windows live wallpaper engine.
