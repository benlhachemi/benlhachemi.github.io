### **Project Summary**

- **Date:** Mar, 2025
- **Description:** A polished, production-ready SaaS landing page template built with Next.js 15, Tailwind CSS v4, and Framer Motion. The page promotes a fictional privacy-friendly web analytics tool (modeled after Pirsch Analytics), featuring animated scroll effects, smooth scrolling, testimonials carousel, pricing tiers, FAQ accordion, and CTA sections - all achieving a 100/100 Lighthouse score.

### Motivation

Portfolio project to demonstrate proficiency with modern frontend tooling - Next.js App Router, Tailwind v4, advanced Framer Motion animations (scroll-triggered, per-character blur/reveal), Lenis smooth scrolling, and shadcn/ui component patterns.

### Links

- **Live Demo:** https://next-saas-lp.vercel.app/
- **GitHub:** https://github.com/benlhachemi/next-saas-lp

### **Tech Stack**

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS v4, tw-animate-css, PostCSS
- **Animation:** Framer Motion (motion v12), Lenis (smooth scroll)
- **UI:** shadcn/ui (New York style), Radix UI primitives
- **Icons:** Lucide React, Tabler Icons React
- **Fonts:** DM Sans (via next/font/google)
- **Linting:** ESLint v9 + eslint-config-next
- **Utils:** clsx, tailwind-merge, class-variance-authority

### **Key Features**

- Animated smooth-scrolling landing page (Lenis + Framer Motion)
- Multiple scroll-triggered animation effects (slide, text blur, text reveal, fade)
- Custom cursor follower effect
- Animated testimonials carousel with auto-play and word-by-word reveal
- Infinite-scroll logo carousel (framework/plugin logos)
- Responsive mobile hamburger menu with full-screen overlay
- Three-tier pricing section with "most popular" highlight and 30-day free trial
- Accordion FAQ section (Radix UI)
- Gradient text effects, custom color theme (green/beige/yellow palette)
- 100/100 Lighthouse score

### **What I Learned**

- Building a full-featured SaaS landing page with Next.js 15 App Router (server/client component split)
- Tailwind CSS v4 configuration and custom design tokens
- Advanced Framer Motion patterns - scroll-triggered animations, spring physics, per-character text effects
- Integrating Lenis for smooth scrolling alongside Framer Motion
- Structuring reusable animation wrapper components (SlideEffect, TextBlurEffect, etc.)
- shadcn/ui setup and customization with Radix UI primitives
- Building an animated testimonials carousel with auto-play and crossfade transitions
