# Souhail Portfolio

Portfolio website built with Next.js and exported statically for GitHub Pages.

## Features
- Simple, clean design
- Responsive layout
- Built with Next.js
- Fully static – no API or server actions

## Getting Started

```bash
# Install dependencies
bun install

# Run the development server
bun run dev
```

## Deployment

The site is statically exported and deployed to Github pages:

```bash
bun run deploy
```

The generated site is placed in the `out` directory (configured via `next.config.js`).
